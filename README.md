# Fantasy Map Complete App

Run locally:

1. npm install
2. npm run dev
3. Open http://localhost:3000

Pages:

- / (home)
- /editor (map editor)
- /viewer (map viewer)
- /instructions, /about, /contact

Export:

- In editor, export JSON (includes map image as data URL) and copy components/ExportedFantasyMapViewer.jsx into other projects.
